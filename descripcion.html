Portafolio Personal

Este portafolio está construido con Go usando el framework Echo para el backend y templ para plantillas HTML type-safe.

El proyecto incluye:

  • Sistema de contacto con protección contra spam (honeypot)
  • Base de datos SQLite para almacenar mensajes
  • Trabajos cron para envío de emails
  • Vista de proyectos con navegación tipo explorador de archivos
